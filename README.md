# Basic Restic TUI

Does the following:
- SFTP 
  - Create repo
  - Creates new snapshot
  - View snapshots
  - Deletes snapshots (prunes as well)
- Local
  - Create repo
  - Creates new snapshot
  - View snapshots
  - Deletes snapshots (prunes as well)

For more information about Restic visit [here](https://restic.net/).
